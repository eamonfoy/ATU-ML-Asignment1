# 1️⃣ Detecting Disinformation Campaigns Using Machine Learning

### Core idea

How machine learning is used to **detect, classify, and track disinformation and propaganda** at scale across social media and news platforms.

### Why this is strong

*   Directly aligned with *Information War*
*   Clear ML pipeline: **data → features → models → evaluation**
*   Lots of reputable academic and industry sources

### Key ML techniques to review

*   **Natural Language Processing (NLP)**
    *   TF‑IDF vs word embeddings (Word2Vec, GloVe)
    *   Transformer models (BERT, RoBERTa) for fake-news classification
*   **Supervised classification**
    *   Logistic Regression, SVMs, Random Forests
*   **Graph ML**
    *   Detecting coordinated behaviour using network analysis
*   **Anomaly detection**
    *   Identifying bot‑like posting behaviour

### Challenges

*   Adversarial adaptation (attackers change language style)
*   Bias in labeled datasets
*   Multilingual disinformation

### Organisations involved

*   Meta (Facebook), Google, X (Twitter)
*   NATO StratCom COE
*   Academic labs (e.g. MIT Media Lab)

### Works well in 3 slides?

✅ Slide 1: Information war + problem definition  
✅ Slide 2: ML techniques pipeline (diagram)  
✅ Slide 3: Challenges + future research

# 2️⃣ Deepfakes and Synthetic Media: ML as Both Weapon and Defence

### Core idea

How **generative models** are used to create deepfakes — and how **ML classifiers** are used to detect them.

### Why this is strong

*   Clear **offence vs defence** framing
*   Very visual (perfect for short presentation)
*   Strong ethical discussion at postgraduate level

### Key ML techniques to review

*   **Generative models**
    *   GANs (StyleGAN, StyleGAN2)
    *   Diffusion models (brief mention)
*   **Deepfake detection models**
    *   CNNs for artifact detection
    *   Frequency‑domain analysis using ML
*   **Multimodal learning**
    *   Cross‑checking audio, video, and lip‑movement

### Challenges

*   Arms race between generators and detectors
*   Dataset freshness (models become obsolete quickly)
*   False positives harming journalism or free speech

### Organisations involved

*   Microsoft (Deepfake Detection Challenge)
*   DARPA (SemaFor program)
*   Academic benchmarks (FaceForensics++)

### Works well in 3 slides?

✅ Slide 1: What deepfakes mean for information warfare  
✅ Slide 2: How GANs & detection models work  
✅ Slide 3: Limitations and future directions
